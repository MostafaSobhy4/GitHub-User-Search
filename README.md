# 🚀 GitHub User Search

---

## 📌 About the Project

A simple and responsive web application that allows users to search for any GitHub profile and display detailed information using the GitHub REST API.

---

## ✨ Features

- 🔍 Search GitHub users by username  
- 👤 Display user profile (avatar, name, bio, location)  
- 📊 Show followers, following, and public repositories  
- ⚡ Real-time data using GitHub API  
- ⏳ Loading and error handling states  
- 🌙 Light / Dark mode toggle  
- 📱 Fully responsive design using Tailwind CSS  

---

## 🛠️ Built With

- HTML5  
- Tailwind CSS  
- JavaScript (ES6+)  
- GitHub REST API
