# 🚀 GitHub User Search

## 📌 Overview
A modern and responsive web application that allows users to search for any GitHub profile and display detailed account information using the **GitHub REST API**.  
The project focuses on API integration, clean UI design, and delivering a smooth user experience.

---

## ✨ Features
- Search GitHub users by username
- Display profile details:
  - Avatar
  - Full name
  - Bio
  - Location
- Show account statistics:
  - Followers
  - Following
  - Public repositories
- Real-time data fetching using GitHub API
- Loading state while fetching data
- Error handling for invalid usernames or failed requests
- Light / Dark mode toggle
- Fully responsive design
- Clean and modern UI built with Tailwind CSS

---

## 🛠️ Tech Stack
- HTML5
- Tailwind CSS
- JavaScript (ES6+)
- GitHub REST API
